# DfM-Ninja

[English version (README.md)](../README.md)

DfM-Ninja は、Dynamics 365 (DfM) のカスタマーサポートエンジニア向けに設計されたプレミアムなフロントエンド専用 SPA (Single Page Application) です。ケース管理、メールテンプレート作成、メタデータ処理、そして Dynamics 365 と LLM 連携ワークフローを橋渡しするクリーンかつ高速なインターフェースを提供します。

## 概要

本アプリケーションは **React 19** と **Vite 7** を使用して構築されており、スタイリングには **Tailwind CSS 4** を採用したモダンなダークテーマ UI を備えています。ブラウザ内のみで動作し、データは **IndexedDB** に永続化されるため、バックグラウンドサーバーは不要です。ケースデータは一括バックアップや JSON 形式での個別エクスポートが可能です。

専用のブックマークレットを介して Dynamics 365 ページと連携し、ケース情報の抽出や Dynamics 環境と Ninja SPA 間のリアルタイムな通信を実現します。

## 主な機能

### 🔍 全文検索 (`Cmd+P` / `Ctrl+P`)
IndexedDB に保存されたすべてのケースを正規表現で検索できます。結果はインタラクティブなアコーディオン形式で表示され、クリックすることで該当するケースやステージへ直接遷移できます。

### 🥷 ブックマークレット連携
ブックマークレットを実行すると Dynamics 365 ページに忍者ボタンが注入されます。これをクリックするとケースメタデータが抽出され、自動的に DfM-Ninja タブにフォーカス（または新規作成）されるため、タブ切り替えの手間が省けます。

### 🛠️ モダンな UI デザイン (v0.6+)
- **2カラムレイアウト**: 左側にプロンプト/ガイドライン、右側に入力フィールドを配置した標準化されたワークフロー。
- **オートセーブ**: 入力した内容はリアルタイムで IndexedDB に保存されるため、ナビゲーション中のデータ紛失を防ぎます。
- **動的コンテンツ**: EJS (Embedded JavaScript) をサポートしており、複雑な条件分岐やフォーマットをテンプレート内で実現可能です。

### 🚀 アップデート通知機能
ユーザーの状態に合わせて以下の通知を自動で表示します：
- **Welcome**: 新規ユーザー向けのオンボーディング。
- **Upgrade**: 以前のバージョンから移行したユーザー向けの案内。
- **Update**: 新機能リリース時の通知 (v0.6.3+)。

### 🔰 変数一覧モーダル
テンプレートで使用可能なすべての変数を参照・ビルドできます：
- **Editable Var.**: ユーザー定義のグローバル変数。
- **Dynamic Var.**: 日付や NC（次回連絡日）の計算・プレビュー。
- **Stage Var.**: 現在の UI から抽出された変数。
- **Sys Temp.**: `attentions` や `teamsDisclaimer` などの登録済みシステムスニペット。

## はじめかた

1. リポジトリをクローンします。
2. 依存関係をインストールします：
   ```bash
   npm install
   ```
3. 開発サーバーを起動します：
   ```bash
   npm run dev
   ```
4. ブラウザで `http://localhost:5178` にアクセスします。

> [!NOTE]
> `src/utils/bookmarkletCode.js` は自動生成ファイルです。`npm run dev` または `npm run build` を実行すると自動的に作成されます。

## ローカル（オフライン）環境での使用

ネットワークのない環境や `file:///` から起動したい場合は、以下のコマンドでパッケージを生成できます：
```bash
npm run build:local
```
これにより、相対パス指定とローカル依存関係（jQuery, EJS）を含んだ `dist.zip` が生成されます。
